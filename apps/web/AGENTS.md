# @9bar/web

Full-stack web application built with TanStack Start, TanStack Router, and React 19.

## Commands

```bash
# Start the dev server (port 3000)
pnpm web dev

# Build for production (typecheck + vite build)
pnpm web build

# Typecheck only
pnpm web typecheck
```

## Architecture

### Routing

Built on **TanStack Router** with file-based, type-safe routing.

- Routes live in `src/routes/`
- The route tree is auto-generated into `routeTree.gen.ts` -- **never edit this file manually**
- Layout routes use the `_` prefix convention (e.g. `_authenticated`, `_unauthenticated`)
- Pathless layout routes group related routes without affecting the URL (e.g. `recipes_/_form/`)
- Private/co-located components use the `-` prefix convention (e.g. `-nav/nav.tsx`, `-form-sections/`)
- Breadcrumb data is attached to routes via `staticData`

### Data Loading

- Server functions via `createServerFn` from `@tanstack/react-start/server`
- Used for SSR data loading in route loaders

### Forms

- TanStack React Form with a custom form hook (`createFormHook`) that pre-registers toolkit field components
- Zod v4 for schema validation

### Localization

- React Aria locale optimization configured for `en-US` and `fr-FR`

### Styling

- Tailwind CSS v4 with a `@source` directive pointing to the toolkit's source files so Tailwind scans toolkit components for class usage
- Custom CSS theme variables defined in `src/styles/globals.css`
- Geist is the primary typeface (loaded from Google Fonts)

## Biome Configuration

This package extends the root Biome config (`"root": false`) and adds:

- Accessibility (`a11y`) recommended rules
- `useExhaustiveDependencies` as an error
- Excludes `*.gen.*` files from linting

## Common Pitfalls

- **Never edit `routeTree.gen.ts`** -- it is auto-generated by TanStack Router. If routes seem stale, restart the dev server.
- **The toolkit does not need to be built** -- `@9bar/toolkit` exports raw TypeScript source. The web app compiles it directly via Vite.
